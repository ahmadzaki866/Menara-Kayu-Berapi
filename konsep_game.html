🧠 Core Gameplay Loop

Inti permainan menggabungkan dua komponen utama: skill platforming (navigasi, timing, refleks) dan pengetahuan Pancasila (menjawab soal kuis). Keduanya sama-sama menentukan kemenangan.

🏃

Navigasi platform, hindari jebakan, raih checkpoint
❓
JAWAB
Soal muncul di checkpoint/box — jawab dalam waktu terbatas
🏆
MENANG
Poin tertinggi / bertahan terakhir / selesaikan tujuan mode

7 Mode Permainan
⚔️
Quiz Race Mode
KOMPETITIF

Balapan platformer + kuis. Siapa yang paling cepat dan paling pintar — dia yang menang.

Pemain: 2–8 orang

Durasi: 5–8 menit

Map: Linear, kiri ke kanan

Mekanik: Semua start bersamaan → hadapi rintangan platform → tiba di Quiz Gate → jawab soal → benar = lanjut, salah = tertahan 3 detik

Menang: Pemain pertama yang mencapai FINISH LINE + skor tertinggi

Soal per game: 5–8 soal checkpoint

Bonus: +5 poin per detik sisa waktu jika jawaban benar

🧩
Puzzle Platform Quiz
STRATEGI

Setiap jawaban menentukan jalur yang bisa dilewati. Salah jalan = jatuh ke jurang.

Pemain: 1–6 orang

Durasi: 8–12 menit

Map: Bercabang (branching paths)

Mekanik: Di tiap persimpangan muncul soal → pilihan A/B/C/D = jalur kiri/kanan/atas/platform yang berbeda → jalur benar membawa lebih banyak koin, jalur salah ada jebakan/rintangan

Menang: Akumulasi poin dari koin yang dikumpulkan + jawaban benar

Soal per game: 10–15 percabangan

Bonus: Jalur tersembunyi dengan 3× koin jika jawaban benar semua

👑
Last Stand Quiz Mode
ELIMINASI

Battle royale edukasi. Jawaban salah = jatuh. Arena makin kecil setiap ronde.

Pemain: 4–8 orang

Durasi: 6–10 menit

Map: Arena terbuka, makin menyempit

Mekanik: Ronde 1: soal mudah → salah = hilang 1 nyawa. Ronde 2: medium → salah = tereliminasi langsung. Ronde 3+: soal sulit, platform bergoyang/jatuh. Pemain yang salah menjawab platform-nya ambruk.

Menang: Pemain terakhir yang masih berdiri

Soal per game: 3–5 soal per ronde, 4–6 ronde

Bonus: Setiap ronde bertahan = +20 poin survival

🤝
Unity Challenge Mode
CO-OP

Mode kerja sama tim — mencerminkan nilai gotong royong Pancasila.

Pemain: 4–8 (2 tim, 2–4 per tim)

Durasi: 10–15 menit

Map: Paralel — dua jalur terpisah, terhubung di titik tertentu

Mekanik:
• Tombol harus diinjak oleh 2 anggota tim bersamaan untuk membuka pintu
• Platform aktif jika semua anggota tim menjawab benar
• Soal tim: satu pemain dapat soal, anggota lain dapat petunjuk (puzzle bersama)
• Pemain bisa "revive" rekan yang jatuh dengan menjawab soal bonus

Menang: Tim pertama yang capai tujuan akhir dengan poin gabungan tertinggi

Bonus: +15 poin jika semua anggota menjawab benar bersamaan

🏁
Checkpoint Quiz Adventure
PETUALANGAN

RPG-lite platformer. Eksplorasi map panjang dengan checkpoint dan soal sebagai kunci progress.

Pemain: 1–6 orang (asinkron — tidak harus bareng)

Durasi: 10–15 menit

Map: Panjang (seperti level Mario), 3 area berbeda

Mekanik:
• 7 checkpoint di sepanjang map
• Di setiap checkpoint → soal wajib muncul
• Benar = save progress, lanjut
• Salah = respawn di checkpoint terdekat (progress tidak hilang)
• Ada item tersembunyi (poin bonus) di platform sulit

Menang: Pemain dengan poin terbanyak saat waktu habis

Soal per game: 7 soal checkpoint + 3–5 soal item bonus

⚡
Speed Quiz Rush
FAST-PACED

Soal kilat, platform bergerak, waktu sempit. Untuk pemain yang suka tantangan intensitas tinggi.

Pemain: 2–8 orang

Durasi: 3–5 menit (mode tercepat!)

Map: Kecil, padat, platform bergerak terus

Mekanik:
• Soal muncul setiap 15 detik, waktu jawab hanya 8 detik
• Platform bergerak — tidak boleh berdiam diri
• Jawaban benar = platform stabil 5 detik + poin
• Jawaban salah = platform makin cepat bergerak
• Soal makin susah tiap menit

Menang: Poin tertinggi saat countdown habis

Soal per game: 15–20 soal cepat

Multiplier: Streak 3 benar = 2× poin, streak 5 = 3× poin

🏆
Team Battle Quiz Mode
TIM VS TIM

2–3 tim saling bersaing. Rebut zona, aktifkan platform, kalahkan tim lain.

Pemain: 4–9 orang (2–3 tim)

Durasi: 10–12 menit

Map: Arena terbagi zona — merah, hijau, biru

Mekanik:
• Setiap zona punya "Quiz Totem" — jawab benar = rebut zona itu
• Zona dikuasai = hasilkan 1 poin/detik untuk tim
• Soal Quiz Totem bisa dijawab siapa saja — maka tim lawan juga bisa merebut balik
• Anggota tim di zona yang sama mendapat bonus pertahanan
• Setiap 3 menit ada "Soal Mega" — poin 3× lipat

Menang: Tim dengan total poin terbanyak saat waktu habis

Soal per game: Terus-menerus (berbasis zona, ~20–30 soal)

🗺️
Desain Map (4 Tema)
🏫 Map 1 — Lingkungan Sekolah

Platform berupa bangku, rak buku, atap gedung, tiang bendera. Warna siang hari cerah. Cocok untuk mode Race dan Adventure.

Elemen unik: Papan tulis interaktif (soal muncul di sini), loker sebagai checkpoint, kantin sebagai area bonus poin

🏡 Map 2 — Desa Nusantara

Platform berupa atap rumah adat, batu sungai, jembatan bambu. Sunset orange. Cocok untuk mode Unity Co-op dan Puzzle.

Elemen unik: Sawah terracing sebagai platform bertingkat, padi emas sebagai koin, rumah adat sebagai shelter sementara

🏙️ Map 3 — Perkotaan Malam

Platform berupa atap gedung, tangga api, crane. Malam hari dengan lampu kota. Cocok untuk Last Stand dan Speed Rush.

Elemen unik: Monas sebagai landmark tengah, platform bergerak berupa taksi terbang, neon sign sebagai petunjuk arah

🏛️ Map 4 — Gedung Negara

Platform berupa tangga gedung MPR/DPR, balkon, koridor. Megah dan formal. Cocok untuk Team Battle.

Elemen unik: Garuda Pancasila raksasa di tengah sebagai landmark, platform berbentuk buku UUD, karpet merah sebagai jalur cepat
